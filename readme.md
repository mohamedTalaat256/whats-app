# What's app
Realtime chat application using react js as frontend and spring boot as backend:


### Backend dependencies:
##### 1- Spring Security
##### 2- JWT
##### 3- Spring data JPA
##### 4- Hibernate Validatior
##### 5- Spring lombok
##### 6- Spring Websocket

##
### Install
open frontend/chat


and run:

```
npm install
```
then:
```
npm start
```
